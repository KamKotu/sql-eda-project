# Global Layoffs Exploratory Data Analysis (SQL Project)

## Overview
This project analyzes a cleaned global layoffs dataset to uncover trends and insights across companies, industries, countries, and time periods.

The analysis focuses on identifying patterns in workforce reductions using SQL.

## Objectives
- Analyze layoffs by company, industry, and country
- Identify yearly and monthly trends
- Detect peak layoff periods
- Rank companies by total layoffs
- Use window functions for advanced analysis

## Tools Used
- MySQL

## Key Analyses Performed

### 1. Company-Level Analysis
Identified companies with the highest total layoffs.

### 2. Industry & Country Trends
Analyzed which industries and countries were most affected.

### 3. Time-Based Trends
- Yearly layoffs
- Monthly layoffs
- Rolling total trends

### 4. Ranking Analysis
Used `DENSE_RANK()` to identify top companies per year.

## Outcome
The analysis revealed clear patterns in global layoffs across industries and time periods, highlighting economic downturn trends and company-level impacts.
